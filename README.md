這個網站主要的功能為追蹤使用者的裝置地理位置，分析後提供有實際意義的資訊，並且找出適合使用者居住的都市環境。

前端：瀏覽器可直接運行<br>
後端： JDK17, Python 3.9, Maven4.0, Spring Boot 3.5, Flask, MySQL 9.3
外部API: Google Geolocation API
